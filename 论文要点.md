# 两篇扩散模型图像水印论文的要点讲解

## 一、两篇论文内容要点讲解

### （一）Tree-Ring Watermarks（NeurIPS 2023）

**1. 核心思想**

Tree-Ring Watermarking提出了一种全新的水印范式——**不修改生成后的图像，而是修改生成过程本身**。传统水印方法都是在图像生成后对其进行后处理修改（如修改频域系数），而Tree-Ring的思路是：在扩散模型采样的**初始噪声向量**中嵌入一个精心构造的图案，让这个图案随着整个去噪过程传播到最终生成的图像中。

**2. 关键技术细节**

- **嵌入位置**：水印被嵌入到初始噪声向量 \(x_T\) 的**傅里叶空间**中，而非直接在像素空间或噪声空间操作。这样做的好处是傅里叶变换具有天然的几何不变性——旋转在像素空间对应傅里叶空间的旋转，平移对应傅里叶系数乘以常数复数，缩放对应傅里叶空间的缩放。
- **三种密钥模式**：
  - **Zeros模式**：在傅里叶空间的圆形掩码区域内将值设为0。该模式对旋转、平移、裁剪和缩放具有完全的数学不变性。
  - **Rand模式**：从高斯分布中采样固定的随机密钥。该模式对生成质量影响最小，且支持多密钥，但对旋转等变换不具不变性。
  - **Rings模式**：在多个同心圆环上赋予恒定值（从高斯分布采样）。该模式在保持旋转不变性的同时，尽可能接近高斯分布以最小化对生成质量的扰动。
- **检测方法**：给定一张待检测图像，通过**DDIM逆向过程**恢复出初始噪声向量 \(\hat{x}\_T\)，然后检查傅里叶空间中预设掩码区域内是否存在预设的密钥图案。检测的统计显著性通过**非中心卡方分布**给出p值。

**3. 主要优势**

- **无需训练**：即插即用，可直接应用于任何现成的扩散模型
- **真正不可见**：因为不修改图像本身，水印在视觉上完全不可感知
- **极强的鲁棒性**：对裁剪、旋转、缩放、JPEG压缩等常见图像变换表现出卓越的鲁棒性
- **可提供严格的统计p值**：检测结果可量化为概率，便于控制误报率

**4. 主要局限**

- 水印容量有限（仅能编码一个固定图案，难以嵌入多位信息）
- 检测需要完整的DDIM逆向过程，计算开销较大
- 对强噪声和极端颜色抖动等攻击的鲁棒性相对有限

### （二）The Stable Signature（ICCV 2023）

**1. 核心思想**

Stable Signature采用了一种完全不同的技术路线——**将水印嵌入能力“根植”于生成模型本身**。具体而言，该方法通过微调潜在扩散模型（LDM）的**解码器（Decoder）** ，使得所有经由该解码器生成的图像都天然携带指定的水印信息。

**2. 关键技术细节**

- **两阶段训练**：
  - **阶段一：预训练水印提取器**。使用HiDDeN框架训练一个水印编码器 \(\mathcal{W}\_E\) 和提取器 \(\mathcal{W}\)，使其能够将 \(k\) 位二进制消息嵌入图像并鲁棒地提取出来。训练完成后丢弃编码器，仅保留提取器。提取器输出经过**PCA白化**处理，以确保对普通图像的输出比特满足i.i.d. Bernoulli分布假设。
  - **阶段二：微调LDM解码器**。固定一个 \(k\) 位二进制签名 \(m\)，在解码器原有的感知损失 \(\mathcal{L}\_i\)（Watson-VGG感知损失）基础上，增加一个消息损失 \(\mathcal{L}\_m\)（二进制交叉熵），联合优化解码器参数。微调仅需约500张图像、100次迭代、单GPU约1分钟。
- **检测方法**：直接使用预训练的水印提取器 \(\mathcal{W}\) 从图像中提取消息 \(m'\)，通过与预设签名 \(m\) 的匹配位数进行假设检验。误报率可通过二项分布精确计算。

**3. 主要优势**

- **与生成过程深度融合**：水印无法通过简单的后处理去除（如注释掉一行代码）
- **支持多用户溯源**：可为不同用户分配不同签名，实现模型级别的身份追踪
- **兼容多种生成任务**：由于只修改解码器，不影响扩散过程，故可兼容文生图、图像编辑、修复、超分等多种LDM-based任务
- **训练高效**：微调只需约500张图像、1分钟GPU时间

**4. 主要局限**

- 需要对生成模型进行微调，不适用于无法修改模型权重的场景（如黑盒API）
- 水印鲁棒性依赖于预训练提取器的质量，对未见过的攻击类型可能失效
- 水印容量受限于提取器的设计（论文使用48位）

### （三）两篇论文的关系

两篇论文虽然都聚焦于扩散模型的图像水印，但代表了两种根本不同的技术哲学：

| 对比维度           | Tree-Ring Watermarks | The Stable Signature |
| ------------------ | -------------------- | -------------------- |
| **水印嵌入时机**   | 采样前（初始噪声）   | 生成中（解码器）     |
| **是否需要训练**   | 无需训练             | 需要微调解码器       |
| **水印与模型关系** | 解耦（即插即用）     | 绑定（根植于模型）   |
| **检测方法**       | DDIM逆向+频域匹配    | 预训练提取器直接提取 |
| **适用场景**       | 黑盒API、已有模型    | 模型发布方自部署     |
| **安全性**         | 依赖密钥保密         | 依赖提取器保密       |

这两篇论文可以视为扩散模型水印领域的**两条并行路线**：Tree-Ring代表“采样时嵌入”路线，Stable Signature代表“模型中嵌入”路线。前者灵活通用但安全性较弱，后者安全牢固但需要模型访问权限。两者的对比正好能够回答一个核心问题：**在AI生成内容版权保护中，水印应该嵌入在“哪里”才是最理想的？**
